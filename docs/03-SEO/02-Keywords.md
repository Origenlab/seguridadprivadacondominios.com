# Keywords y densidad

## Keyword principal
**`seguridad privada para condominios`**

## Variantes semánticas (intercambiar para no sobreoptimizar)
- seguridad residencial
- vigilancia para condominios
- protección de condominios
- seguridad para fraccionamientos
- seguridad para residenciales
- vigilancia 24/7
- control de acceso para condominios

Todas por debajo del 2% (umbral seguro anti-stuffing).

## Title HTML — patrón
```
{Título único} | SEPRICO {Ubicación}
```
Máx 60 chars. Ejemplo: `Seguridad Privada para Condominios | SEPRICO CDMX y Edomex`

## Meta description — patrón
Máx 160 chars. Incluir keyword 1 vez. Atractiva, no spam.

## Anchor text en links internos
NO usar siempre "click aquí" o "más info". Usar texto descriptivo:
- ✅ "Conoce nuestros servicios de seguridad privada para condominios"
- ❌ "Click aquí"
